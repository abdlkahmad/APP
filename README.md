# Store Web App

This project provides a simple Android application wrapping the website [store.abdulkareem.net](https://store.abdulkareem.net) in a WebView.

## Building

The environment does not include the Android SDK or network access to download dependencies, so building the APK cannot be completed here.

To build the APK on your own machine:

1. Install Android Studio with SDK API 34 or later.
2. Open the project in Android Studio.
3. Allow Gradle to download dependencies.
4. Build the `app` module (`Build > Build Bundle(s) / APK(s) > Build APK(s)`).

The application targets Android API 34 and supports Android 5.0 (API 21) and above.
